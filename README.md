## TowerDefense-Game-Mechanic

This prototype mechanic for Tower Defense was developed by me to test my Unity development skills.
In this mechanic, I tried to use a lot of techniques, patterns and assets, so you need to start viewing from the "Startup" scene.
This project reveals my ability to write competent architectural code and knowledge of optimization and the Unity engine itself.

## Dependencies

*All plugins are free and available on Unity Assets Store*

* [Text Mesh Pro](https://docs.unity3d.com/Manual/com.unity.textmeshpro.html)
* [Addressables](https://docs.unity3d.com/Manual/com.unity.addressables.html)
* [Analytics Library](https://docs.unity3d.com/Packages/com.unity.analytics@3.6/manual/index.html)
* [JetBrains Rider](https://docs.unity3d.com/Packages/com.unity.ide.rider@2.0/manual/index.html)
* [Multiplayer HLAPI](https://docs.unity3d.com/Packages/com.unity.ide.rider@2.0/manual/index.html)
* [Universal RP](https://docs.unity3d.com/Packages/com.unity.render-pipelines.universal@10.7/manual/index.html)
